# ULPM Online Store

A responsive web store concept connecting local farmers and customers through homemade agricultural products.

## About the Project

ULPM Online Store is a static frontend project designed as an online marketplace for local farmers and customers.  
The platform presents homemade agricultural products, promotes local production, and allows users to explore product categories such as vegetables, dairy products, fruit, meat, honey, winter preserves, and ready-made baskets.

The idea behind the project is to create a simple, transparent, and user-friendly shopping experience for people who want to buy directly from local producers.

## Features

- Landing page for the local farmers association
- Information sections about production, tradition, and platform purpose
- Customer access to the online store
- Product category navigation
- Product cards with prices, producer name, details, and cart actions
- Discount and promotional sections
- Responsive layout using Bootstrap

## Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- Bootstrap Icons

## Pages

- `index.html` – main landing page
- `prodavnica.html` – online store/product listing page

## Project Status

This project is currently a static frontend prototype.  
Some interactive features such as login, cart functionality, product details, and checkout are planned as future improvements.
